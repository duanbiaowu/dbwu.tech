如何获取更多的免费服务器

本文从开发者和个体创业者的角度，教你如何获取更多的免费服务器资源，翻译一下，就是教你 “如何白嫖”。

免费服务器的好处

  • 最重要的：免费
  • 随意折腾，不用担心搞坏了，例如你可以任性执行 rm -rf /
  • 学习云计算服务商的成熟管理方式
  • 方便开发和测试，例如执行一些基准测试、模型训练
  • 异构环境的解决方案，可以节省下大量的时间，虽然容器也可以实现，但是容器受制于开发机环境配置，毕竟有时候需要一个干净纯粹的测试环境来执行一些 “骚操作”

针对创业者

主要介绍 3 家产品，分别是 AWS, Google Cloud, Microsoft Azure, 当然，虽然服务器是免费使用的，但是并不意味着没有门槛，这个门槛就是你需要是一个创业者，换言之，你需要有自己的公司。此外，申请服务时还需要填写支付等信息 (例如 Visa/地址信息/…)。

AWS

申请成功后，最低有 $2500 美刀的使用券，最高有 $100000 美刀的使用券，使用期限为 6 个月。

申请链接： https://aws.amazon.com/startups/

Google Cloud

申请时需要选择所在地区和项目类型，不同的配置获取到配置和使用时长是不同的，适合快速做出原型后，投入到市场验证是否可行。

申请链接： https://startup.google.com/programs/

Microsoft Azure

相比另外两家，除了可以申请到主机外，还有可能申请到 OpenAI 使用额度 (免费使用 ChatGPT)。

申请链接： https://www.microsoft.com/en-us/startups

针对普通开发者

如果你还不是创业者，那么可以选择的就不多了 (国内的云服务计算提供商就别想了)，不过还有 Google Cloud 提供的免费服务器。

申请链接： https://shell.cloud.google.com/

申请完成后，进入初始化和配置时间，第一次等待的时间可能稍微有点长 (大概 1 分钟左右)。
